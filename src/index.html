Info about the AidData dataset

The AidData dataset contains information about aid donations between two countries.

Each row represents one transaction between two countries with the following attributes:

  • Year: year of the commitment
  • Donor country providing the financial resource
  • Recipient: country or organization receiving the money
  • Commitment Amount USD: the total amount of financial resources provided
  • Coalesced Purpose Name: the purpose of the transaction

Visualization 1

Create an overview of the relationships between countries so that it is possible to see who donates to whom and how much.

  • Who are the major donors and to which countries do they donate the most? And conversely, who are the major receivers and which countries do they receive from the most?
  • Are there groups of countries that tend to donate/receive to/from a similar set of countries?

Visualization 3

Considering only the top 5 purposes of donation:

  • How does the relationship between countries look like in terms of purposes?
  • What purposes do countries donate for to other countries?
  • Are there countries that donate to a given country using multiple purposes? Or do countries always donate using one single purpose when donating to another country?
